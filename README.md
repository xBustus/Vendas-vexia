# Vexia — Site institucional

Plataforma inteligente para gestão de coleta urbana e frotas.

Este repositório contém a landing page de vendas do **Vexia**, hospedada via **GitHub Pages**.

## Estrutura

```
Vendas-vexia/
├── index.html              # Página principal (antiga aboutVexia.html)
├── vexiaContato.html       # Página de contato
├── vexiaTermos.html        # Termos de uso
├── vexiaPrivacidade.html   # Política de privacidade
└── assets/
    ├── vexia.png           # Logo / favicon
    └── Profile-Picture.ico # Ícone original (backup)
```

## Dependências externas (CDN)

- [Tailwind CSS](https://cdn.tailwindcss.com)
- [Lucide Icons](https://unpkg.com/lucide@latest)
- [Leaflet](https://unpkg.com/leaflet@1.9.4) — mapa interativo na home
- [Google Fonts — Inter](https://fonts.google.com/specimen/Inter)

Nenhuma dependência local — basta servir os HTMLs estaticamente.
